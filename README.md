# Hey, I'm Shayan 👋

I work at the intersection of **community and data**. During the day I moderate a 40K member subreddit for a US telecom brand and build the technical infrastructure behind it. Pipelines that capture every post and comment in real time. AI layers that classify sentiment and intent automatically. Dashboards that surface what a community is actually saying.

Most of what I build lives in Google Apps Script, Zapier, and the Reddit and OpenAI APIs.

---

## What I Build

```
Reddit Community → Data Pipeline → AI Analysis → Dashboard
```

- **Automated pipelines** that pull, clean, and structure community data in real time
- **AI classification layers** that tag posts by sentiment, intent, topic, and customer type
- **Custom dashboards and web apps** built on top of Google Sheets and Apps Script
- **Subreddit infrastructure** including wikis, automod rules, and moderation workflows

## Projects

### 🔍 Subreddit Analysis Pipeline
> Pull historical Reddit data via Arctic Shift API, structure it with Apps Script, run sentiment and keyword analysis, and surface insights through a custom dashboard.

Built to answer the question most community managers cannot: *what is actually happening in this subreddit over time, not just today.*

**Stack:** Arctic Shift API · Google Apps Script · Google Sheets · HtmlService · JavaScript

---

### ⚡ Zapier Data Pipeline Showcase
> A documented collection of Zapier workflow patterns, webhook triggers, scheduled API pulls, multi-step filter and formatter flows, and an end-to-end pipeline where Zapier hits a deployed Apps Script web app that writes structured data into Sheets.

Most people use Zapier for simple automations. This repo shows what it looks like when you treat it as a proper data routing layer.

**Stack:** Zapier · Google Apps Script · Google Sheets · Webhooks · REST APIs
